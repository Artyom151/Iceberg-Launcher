Всем привет! Это мой первый лаунчер майнкрафт на **Node JS**!

Чтобы установить и запустить надо скачать **Node JS** и **Java** и выполнить эти команды:
1.
```bash
npm install
```
2.
```bash
npm start
```

Вот и всё! :3
